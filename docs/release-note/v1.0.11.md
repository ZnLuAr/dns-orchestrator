# v1.0.11 发行说明

## 新功能

### 账户管理

- **独立账户管理页面**：新增完整的账户管理页面，支持账户的增删改查、导入导出，集中管理所有 DNS 服务商账户

### 侧边栏导航

- **树形结构导航**：侧边栏从平铺列表改为树形结构，账户和域名分层展示，导航更清晰
- **响应式折叠**：中等屏幕尺寸下侧边栏自动折叠为图标模式，鼠标悬停时浮层展开，不挤压主内容区
- **移动端优化**：改进移动端侧边栏的交互体验

### 工具箱增强

- **DNS 查询自定义服务器**：支持选择系统默认、常用公共 DNS（Google、Cloudflare、阿里、腾讯等）或输入自定义 DNS 服务器进行查询，并显示实际使用的服务器信息
- **IP 查询服务升级**：切换至 ipwho.is 服务，支持 HTTPS，为后续 iOS 平台上架做准备
- **标签页动画**：工具箱标签页切换添加滑动指示器动画，交互更流畅

### DNS 管理

- **移动端批量选择**：DNS 记录卡片支持批量选择模式，移动端也能多选删除

## 改进

- 设置页面调试模式选项仅在开发环境显示，生产构建不包含调试代码
- IP 查询错误提示优化，月度限额等错误以中文友好展示
- 改进 Sheet 组件的无障碍访问支持

## 依赖升级

- i18next、@vitejs/plugin-react、vite 等前端依赖更新
- base64ct、cc、dlopen2、flate2 等 Rust 依赖更新

## 升级说明

从 v1.0.10 升级无需额外操作，直接安装新版本即可。
