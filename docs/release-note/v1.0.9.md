# v1.0.9 发行说明

## 新功能

### 工具箱功能增强

- **SSL 证书检查**：支持检查网站的 SSL/TLS 证书信息，包括有效期、颁发者、SAN、证书链等，支持自定义端口，自动检测 HTTP/HTTPS 连接状态
- **IP 地理位置查询**：支持 IP 地址或域名查询，域名自动解析显示所有关联 IP 的地理位置，包括国家/地区、城市、ISP、AS 号等信息
- **工具箱页面重构**：采用标签页导航，切换更便捷

### DNS 管理

- **批量删除功能**：支持多选 DNS 记录并一键批量删除，支持全选/取消全选操作

## 平台改进

### Android

- **APK 安装器插件**：新增 `tauri-plugin-apk-installer` 自定义插件，使用 FileProvider 正确处理 Android 7.0+ 的 URI 转换，解决高版本系统 APK 安装兼容性问题

## 依赖升级

- Biome 升级至 v2.3.8
- React 升级至 v19.2.1
- 新增 `@radix-ui/react-collapsible` 和 `@radix-ui/react-tabs` 组件

## 升级说明

从 v1.0.8 升级无需额外操作，直接安装新版本即可。
