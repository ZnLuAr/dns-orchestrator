# v1.4.2 发行说明

## 概述

v1.4.2 是一个 UI 一致性优化版本，主要通过创建通用布局组件统一了各页面的样式规范，并修复了系统主题跟随和桌面端交互体验问题。

## 新功能

### 通用布局组件

新增 6 个通用 UI 组件，统一页面布局和样式：

- **PageLayout**：页面布局容器，统一 flex 布局结构
- **PageHeader**：页面标题栏，统一 header 样式和移动端菜单
- **PageContainer**：内容区容器，统一 padding 和滚动行为
- **EmptyState**：空状态组件，支持 compact/default/large 三种尺寸
- **SectionCard**：带标题的卡片组件，统一 Card 使用方式
- **SettingSection**：设置页专用组件，包含 Section/Item/Row 三个子组件

## 改进

### UI 一致性统一

- **间距规范**：页面容器统一 `p-4 sm:p-6`，Header 统一 `gap-2 sm:gap-3`
- **字体层级**：页面标题统一 `text-xl`，Card 标题统一 `text-lg`
- **空状态样式**：页面级 `py-16` 带图标，表格内 `py-8` 无图标
- **设置页图标**：添加设置图标，与其他页面保持一致

### 桌面端体验优化

- **禁止文字选择**：全局禁用文字选择，保持桌面应用原生体验
- **表单元素例外**：input、textarea 等表单元素保持可选择

### 系统主题跟随

- **实时监听**：修复系统主题变化时应用未自动切换的问题
- **即时响应**：系统主题切换后，应用立即跟随变化，无需重启

## 重构

以下页面使用新组件进行了重构：

- SettingsPage
- AccountsPage
- DomainSelectorPage
- HomePage
- DnsRecordPage
- ToolboxPage

## 升级说明

从 v1.4.1 升级无需额外操作，直接安装新版本即可。
