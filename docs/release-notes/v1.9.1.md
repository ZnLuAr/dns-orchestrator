# v1.9.1 发行说明

## 概述

v1.9.1 是一个维护性修复版本。本版本修正了 v1.9.0 中 CI/CD 流水线的 Android 和 iOS 构建路径问题，确保 Release 流水线能正确完成 APK 签名和 IPA 上传；同时升级了 wry 依赖版本。

## Bug 修复

### 🔧 Release 流水线修正

修复 v1.9.0 架构重构后 CI/CD 流水线中遗留的路径和参数问题：

**路径修正**：
- Android 构建输出路径从 `src-tauri` 修正为 `dns-orchestrator-tauri`
- iOS 构建输出路径同步修正
- 影响范围：Keystore 解码、APK 重命名、APK 签名、Release 上传、IPA 查找

**参数修正**：
- 修正 `pnpm tauri android build` 的 `--apk` 参数（移除多余的 `true`）
- 简化 `update-latest-json` job 的 `needs` 依赖语法
