# Toolbox - 已完成功能

以下功能已实现并可用。

## 核心诊断工具（优先级高）

### ✅ DNS 查询

**用途**: 查询域名的 DNS 记录
**场景**: 检查域名解析是否正常
**支持记录类型**: A, AAAA, CNAME, MX, TXT, NS, SOA 等
**实现**: 使用 hickory-resolver 库

### ✅ WHOIS 查询

**用途**: 查询域名注册信息
**场景**: 查看域名所有者、注册商、到期时间
**实现**: 通过 WHOIS 服务器查询

### ✅ SSL 证书检查

**用途**: 检查网站的 SSL/TLS 证书
**场景**: 验证证书有效性、到期时间、证书链
**实现**: TCP TLS 连接 + 证书解析
**特性**:
- 证书链完整性检查
- 有效期验证
- 自定义端口支持

### ✅ IP 地理位置查询

**用途**: 查询 IP 地址的地理位置和运营商信息
**场景**: 分析访问来源、CDN 节点位置
**实现**: 使用 ip-api.com API

### ✅ HTTP 头检查器

**用途**: 显示 HTTP 响应头 + 安全头分析
**场景**: 检查缓存策略、安全配置、服务器信息
**实现**: Reqwest HTTP 客户端
**特性**:
- 自定义请求头
- 安全头分析（HSTS, CSP, X-Frame-Options 等）
- 支持多种 HTTP 方法

### ✅ DNS 传播检查

**用途**: 从全球多个 DNS 查询，检测传播状态
**场景**: 验证 DNS 记录是否已全球同步
**实现**: 并发查询多个公共 DNS 服务器
**特性**:
- 全球 DNS 服务器（Google, Cloudflare, OpenDNS 等）
- 传播状态分析
- 不一致记录检测

### ✅ DNSSEC 验证

**用途**: 检查 DNSSEC 启用状态和密钥信息
**场景**: 验证域名安全配置
**实现**: 查询 DNSKEY, DS, RRSIG 记录
**特性**:
- 验证链完整性
- 密钥信息展示
- 签名验证状态
